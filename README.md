# FinAssist 💰

Assistente de controle financeiro pessoal com Inteligência Artificial.

O projeto permite registrar gastos mensais, calcular saldo e receber
uma análise automática feita por um modelo de IA local.

## Tecnologias utilizadas

- Python
- Streamlit
- Ollama
- Llama 3
- Pandas

## Funcionalidades

- Registro de gastos
- Cálculo automático de saldo
- Tabela de despesas
- Gráfico de gastos
- Análise financeira feita por IA

## Como executar o projeto

1. Instale as dependências

pip install -r requirements.txt

2. Execute o Streamlit

streamlit run main.py

3. Abra no navegador

http://localhost:8501

## IA utilizada

A análise financeira é feita utilizando o modelo Llama 3 rodando localmente
através do Ollama.
